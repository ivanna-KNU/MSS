# Лабораторна робота 2: Побудова лінійної моделі з допомогою псевдообернених операторів

## Мета роботи

Побудова лінійної моделі, яка перетворює вхідне зображення x2.bmp у вихідне зображення y5.bmp за допомогою псевдообернених операторів.

## Вхідні дані

- Вхідне зображення: x2.bmp (188×256 пікселів, градації сірого)
- Вихідне зображення: y5.bmp (188×181 пікселів, градації сірого)

## Метод вирішення

Використано формулу A = YX⁺, де:
- X - матриця вхідних сигналів (кожен стовпець - це вектор пікселів одного стовпця зображення)
- Y - матриця вихідних сигналів
- X⁺ - псевдообернена матриця (обчислена за формулою Мура-Пенроуза через numpy.linalg.pinv)
- A - матриця лінійного оператора

До матриці X додано рядок одиниць знизу згідно з формулою (1) з методички.

## Результати

Розміри матриць:
- X: 257×188 (256 рядків пікселів + 1 рядок одиниць, 188 стовпців)
- Y: 181×188 (181 рядок пікселів, 188 стовпців)
- A: 181×257 (матриця оператора)

Похибка дорівнює нулю, що є очікуваним результатом для лінійної моделі з псевдооберненою матрицею - модель точно відтворює вихідні дані.
