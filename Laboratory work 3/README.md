# Лабораторна робота 3

## Мета роботи

Оцінити невідомі параметри системи коливань трьох мас (жорсткості пружин c₁, c₂, c₃) за допомогою функцій чутливості, використовуючи експериментальні дані спостережень стану системи.

## Виконана робота

Реалізовано програму на Python, яка виконує параметричну ідентифікацію методом функцій чутливості. Програма включає:

- Читання експериментальних даних з файлу y10.txt (6 координат стану, 251 точка часу)
- Побудову матриці системи диференціальних рівнянь dy/dt = Ay
- Обчислення похідних матриці A по параметрах β = (c₁, c₂, c₃)ᵀ
- Чисельне інтегрування системи стану методом Рунге-Кутта 4-го порядку
- Інтегрування системи матриць чутливості dU/dt = AU + ∂(Ay)/∂βᵀ
- Обчислення поправки параметрів Δβ через інтеграли від матриць чутливості
- Ітеративний алгоритм покращення параметрів з обчисленням критерію якості I(β)

## Отримані результати

Алгоритм збігся за 7 ітерацій. Знайдені значення параметрів:

- c₁ = 0.1400
- c₂ = 0.3000
- c₃ = 0.2000

Критерій якості зменшився з початкового значення 77.09 до фінального I(β) = 1.07×10⁻⁸, що вказує на високу точність ідентифікації параметрів.

Початкове наближення було β₀ = (0.1, 0.1, 0.4)ᵀ, але алгоритм успішно знайшов оптимальні значення, які мінімізують різницю між спостережуваними та моделюваними даними.
