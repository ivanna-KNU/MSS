# Лабораторна робота 1

## Мета роботи

Визначити модель функції y(t) = a₁t³ + a₂t² + a₃t + Σ aᵢ sin(2πfᵢ₋₃t) + aₖ₊₁ для спостережуваної дискретної функції з файлу f10.txt, використовуючи дискретне перетворення Фур'є для знаходження частот.

## Виконана робота

Було реалізовано програму на Python, яка виконує наступні кроки:

1. Читання даних з файлу f10.txt (501 значення)
2. Самостійна реалізація дискретного перетворення Фур'є згідно формули cₓ(k) = (1/N) Σ x(m)e^(-i2πkm/N)
3. Визначення домінуючих частот з DFT спектру (вибрано топ-5 найбільших амплітуд)
4. Побудова системи рівнянь для методу найменших квадратів
5. Розв'язання системи та знаходження коефіцієнтів моделі

## Отримані результати

Знайдені домінуючі частоти:
- f₁ = 0.2 Гц
- f₂ = 0.4 Гц
- f₃ = 0.6 Гц
- f₄ = 0.8 Гц
- f₅ = 15.0 Гц

Коефіцієнти моделі:
- a₁ = -8.63 (коефіцієнт при t³)
- a₂ = 71.21 (коефіцієнт при t²)
- a₃ = -123.45 (коефіцієнт при t)
- a₄ = 56.46 (коефіцієнт при sin(2π·0.2·t))
- a₅ = 6.40 (коефіцієнт при sin(2π·0.4·t))
- a₆ = 1.57 (коефіцієнт при sin(2π·0.6·t))
- a₇ = 0.47 (коефіцієнт при sin(2π·0.8·t))
- a₈ = 14.98 (коефіцієнт при sin(2π·15.0·t))
- a₉ = -2.24 (константа)

Похибка апроксимації: RMSE = 3.51
